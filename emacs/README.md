# pi-emacs-bridge.el

Emacs client for `extensions/emacs-bridge.ts`.

Main commands:

- `M-x pi-emacs-bridge-attach`
- `M-x pi-emacs-bridge-detach`
- `M-x pi-emacs-bridge-send-buffer`
- `M-x pi-emacs-bridge-send-region`
- `M-x pi-emacs-bridge-send-position-dwim`
- `M-x pi-emacs-bridge-send-prompt`
- `M-x pi-emacs-bridge-send-prompt-with-location-dwim`
- `M-x pi-emacs-bridge-send-error-at-point`
- `M-x pi-emacs-bridge-send-return`
- `M-x pi-emacs-bridge-send-escape`
- `M-x pi-emacs-bridge-clear-editor`
